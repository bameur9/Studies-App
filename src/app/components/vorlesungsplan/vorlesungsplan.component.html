    - Wie lege ich einen Vorlesungsplan an ?
    (zb. Math am Dienstag um 09:45 bis 11h15)

    1. klicken Sie auf das Symbol (+) oder klicken Sie auf das Feld der Uhrzeit des Unterricshtsbeginns.

    2. Das untere angezeigte Bild öffnet sich und Sie müssen Ihre Daten eingeben.
    Hinweis: Es ist sehr wichtig, dass Sie das Feld repeat markieren, wenn Sie möchten, dass sich Ihr Programm jede Woche wiederholt. folgen Sie der folgenden Anweisung:
    Markieren Sie das Kästchen "repeat" -> wählen Sie "Weekly" -> "save".
